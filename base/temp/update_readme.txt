Base64网络传输客户端更新包
版本: 2.0.0

更新内容:
1. 修复GUI模式下自动更新问题
2. GUI界面现在不会自动检查和下载更新
3. 在GUI模式下发现新版本时，只会提示用户手动更新
4. 保持命令行模式的自动更新功能不变

安装说明:
1. 备份当前客户端程序
2. 将新的client程序替换原有程序
3. 重新启动客户端

注意事项:
- 此版本专门针对GUI模式优化
- 命令行模式功能保持不变
- 建议在更新前关闭正在运行的客户端

技术改进:
- 添加了gui_mode_active全局变量来区分运行模式
- 优化了版本检查逻辑，避免GUI模式下的自动更新
- 改善了用户体验，让用户可以主动控制更新时机
