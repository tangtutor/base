Base64网络传输客户端更新包 v2.0.0

更新内容:
- 升级到版本2.0.0
- 新增--gui参数启动图形界面
- 改进命令行参数处理
- 支持独立的update命令检查更新
- 优化用户界面和交互体验

使用说明:
1. 使用 './client update' 检查并下载更新
2. 使用 './client --gui' 启动图形界面
3. 默认为命令行模式，需要--gui参数才启动图形界面

安装说明:
1. 停止当前客户端
2. 解压此更新包
3. 替换客户端可执行文件
4. 重新启动客户端

注意事项:
- 此更新包将客户端版本设置为2.0.0
- 图形界面需要使用--gui参数启动
- 确保与服务器版本保持一致
